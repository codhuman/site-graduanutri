Comando pwd

O que é o comando pwd?

O comando pwd é um comando que utilizamos para descobrir o caminho completo do diretório em que o usuário que estamos utilizando está no momento em que digitamos o comando.

O comando pwd é bastante útil em três momentos:

  • Quando o caminho do diretório em que estamos é muito grande e queremos copiá-lo para algum parâmetro de um outro comando que estamos utilizando.
  • Quando navegamos por tantos diretórios que nem lembramos mais direito onde estamos.
  • Quando queremos que um script obtenha e utilize o caminho completo do diretório atual em algum processamento interno.

Sintaxe do comando pwd

A sintaxe do comando pwd é composta por dois elementos:

  • O comando pwd
  • Opções

O comando pwd pode ser utilizado sem nenhuma opção ou com uma das quatro opções disponíveis:

  • -L - Exibe o link simbólico que faz referência à pasta atual do usuário
  • -P - Exibe o caminho real (físico) da pasta atual do usuário
  • --help - Exibe a ajuda do comando pwd
  • --version - Exibe a versão do comando pwd

O comando pwd sem nenhuma opção funciona da mesma forma que o comando pwd -L.

Lista de Opções do pwd

O comando pwd possui as 4 opções listadas abaixo:

  • -L -> Logical – Caminho Lógico
  • -P -> Physical – Caminho Físico
  • --help -> Ajuda
  • --version -> Versão

pwd -L

O comando pwd utilizando a opção –L exibe o link simbólico que faz referência à pasta atual do usuário. Um link simbólico é um atalho para uma pasta ou arquivo e é criado com o comando ln –s.

pwd -P

O comando pwd utilizando a opção –P exibe o caminho físico da pasta atual do usuário. Na figura abaixo, podemos ver a diferença entre o comando utilizando a opção –L e –P.

pwd “sem opção”

O comando pwd sem nenhuma opção funciona da mesma forma que o comando pwd –L, ou seja, exibe o link simbólico que faz referência à pasta atual do usuário.

pwd --help

O comando pwd utilizando a opção --help exibe a ajuda do comando pwd. A opção --help é comum a quase todos os comandos do Linux, senão todos, e funciona como um mini manual do comando.

No caso do comando pwd, a opção --help exibe a descrição do comando, a lista de opções, como funciona o default (pwd sem nenhuma opção) e também o “Exit Status”. O “Exit Status” é o estado de saída do comando após a sua execução, que no caso, retorna zero quando comando pwd é executado com sucesso.

pwd --version

O comando pwd utilizando a opção --version exibe a versão do comando pwd juntamente com outras informações, tais como:

  • O Copyright – direitos de cópia
  • O tipo de Licença, que neste caso é do tipo GPLv3+
  • Orientações sobre software livre
  • O nome do autor do comando pwd, que neste caso é Jim Meyering
